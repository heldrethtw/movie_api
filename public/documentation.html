Movie API Documentation

The Donkey Archive

The API is designed to be intuitive and easy to use, providing endpoints for searching films, retrieving detailed information about a specific title, and exploring films by various categories, including directors, genres, and release years. Whether you're looking to enrich a film-related project, conduct in-depth cinematic research, or simply explore the diverse landscape of art cinema, the Donkey Archive API is your go-to source for comprehensive and reliable film data.

Movie Endpoints

Request Method URL Response Description
GET /api/movies return index of all movies to user array of movie objects
GET /api/movies/{id} return a specific movie to user movie object by id
GET /genres/{name} return data about a genre to user genre object by name
GET /directors/{name} return data about a director to user director object by name
GET /years/{year} return data about a year to user year object by year
GET /search/{term} return data about a search term to user search object by term

User Endpoints

Request Method URL Response Description
