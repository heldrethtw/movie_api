Movie API Documentation

The Donkey Archive

The Donkey Archive API provides endpoints for efficient access to movie-related data. These endpoints allow users to search for movies, retrieve detailed information about specific titles, and explore films by directors, genres, and release years. This API offers a reliable source of comprehensive film data for various applications.

Movie Endpoints

Request Method URL Response Description
GET /api/movies return index of all movies to user array of movie objects
GET /api/movies/{id} return a specific movie to user movie object by id
GET /genres/{name} return data about a genre to user genre object by name
GET /directors/{name} return data about a director to user director object by name
GET /years/{year} return data about a year to user year object by year
GET /search/{term} return data about a search term to user search object by term

User Endpoints

Request Method URL Response Description
POST /user/register allow new user to register registration conirmation or error
POST /user/login allow user to login login confirmation or error
POST /user/logout allow user to logout logout confirmation or error
GET /user/{id} return user data user object by id
PUT /user/{id} allow user to update their data user object by id
DELETE /user/{id} allow user to delete their account user object by id
POST /user/{id}/favorites allow user to add a favorite movie confirmation
DELETE /user/{id}/favorites allow user to remove a favorite movie confirmation
GET /user/{id}/favorites return user's favorite movies array of movie objects
