Movie API Documentation

The Donkey Archive

The API is designed to be intuitive and easy to use, providing endpoints for searching films, retrieving detailed information about a specific title, and exploring films by various categories, including directors, genres, and release years. Whether you're looking to enrich a film-related project, conduct in-depth cinematic research, or simply explore the diverse landscape of art cinema, the Donkey Archive API is your go-to source for comprehensive and reliable film data.
